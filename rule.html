わいくらサーバー 利用規約

改定日：2025年10月12日

第1章 総則

第1条（目的）

本利用規約（以下「本規約」）は、「わいくらサーバー」（以下「当運営」）が提供するマインクラフトサーバーおよび関連サービス（以下「本サービス」）の利用条件を定めるものです。ユーザーは、本規約に同意のうえ本サービスを利用するものとします。

第2条（適用範囲）

本規約は、ユーザーと当運営との間に成立するすべての関係に適用されます。当運営は、必要に応じて本規約を改定でき、改定後の規約はサーバー内または公式 Discord にて告知された時点から効力を生じます。

第3条（準拠法および管轄）

本規約の準拠法は日本法とします。本サービスに関して紛争が生じた場合、当運営代表の所在地を管轄する裁判所を専属的合意管轄とします。

第2章 サービス運営

第4条（運営の権限）

  • 当運営は、本サービスの維持・安全確保のため、ユーザーの行動に対して制限、警告、停止、削除等の措置を行う権限を有します。
  • 当運営は、事前通知なく本サービスの内容を変更、追加、廃止できるものとします。
  • 当運営は、本サービスの利用により生じた損害・不利益について一切責任を負いません。
  • 技術的または運用上の理由により、サーバーを一時停止または終了できるものとします。

第3章 ユーザーの義務

第5条（利用資格）

  • Minecraft 正規版（Mojang/Microsoft 認証済みアカウント）が必須。
  • 認証済みクライアント（Modrinth App、GDLauncher、Prism Launcher、Lunar Client 等）は原則使用可。ただし不正改変や不正機能は禁止。
  • Discord 等外部サービス利用時は、当該サービス規約にも従う。
  • 未成年者は法定代理人の同意が必要。

第6条（禁止行為）

ユーザーは以下の行為を行ってはなりません。違反時、警告やBAN等の措置があります。

第1項 ゲーム内行為
  • チート・ハック・不正改造クライアント・バグ悪用・外部ツール不正使用（例：X-ray、オートクリッカー、マクロ、KillAura、Fly、SpeedHack 等）
  • 公認規格施設・他プレイヤー所有物の破壊・編集・妨害
  • サーバー運営妨害目的の負荷建築や装置作成
  • AFK回避の自動操作（AFKプール等）
  • サーバー運営やゲーム進行を妨げる行為
第2項 コミュニケーション・表現
  • 不特定多数へのメンションや連続スパム
  • 過度な挑発、脅迫、ハラスメント、人格攻撃
  • 差別的発言やヘイト表現
  • ポルノ・過度な暴力描写を含むコンテンツ投稿
  • なりすまし行為
  • チャンネル用途にそぐわない投稿
共通禁止事項
  • 金銭・ギフト券・電子マネーを用いたアイテム・役職・国籍等の取得
  • 本サービスを用いた営利活動
  • Mojang/Microsoft EULA、Minecraft サーバー運営ガイドライン違反
  • ナチス関連シンボル・名称・建築・スキン等の表示
  • 人種・性別・宗教・国籍等による差別・侮辱・憎悪表現
  • 不正アクセス、サーバーデータ改変、攻撃等
  • 公式Discordサーバーに参加せずサーバーに参加

第7条（規約違反への対応）

当運営は違反行為に応じて適切な措置を講じます。緊急性や再発防止の観点から、予告なく即時措置を行う場合があります。

第4章 国家・ゲームシステム

第8条（国家運営）

  • すべての国家・自治体は「国際連合」傘下に属し、秩序・発展を目的とします。
  • 国家名・都市名は日本語で統一。
  • 国家制度・仕様は運営判断で変更・リセット可能。
  • 国家間紛争・外交問題は当運営が最終決定権を有します。

第5章 データ・知的財産

第9条（データ管理）

サーバー上の建築物、国家、データ等は当運営が管理・保存・削除権限を有します。バックアップ・復旧は保証されません。

第10条（知的財産）

「Minecraft」は Microsoft および Mojang Studios の登録商標です。当運営は非営利のファン活動として運営され、これらの権利とは無関係です。

第6章 プライバシー

第11条（ユーザー情報の取り扱い）

Discord ID・Minecraft ID 等を管理目的のみに使用し、プライバシーポリシーに基づき適切に管理します。

第7章 免責

第12条（免責）

サーバー停止、データ破損、通信障害その他トラブルによる損害について、一切責任を負いません。本サービスは「現状有姿」で提供され、動作保証はありません。

第8章 問い合わせ

第13条（問い合わせ）

本規約や運営に関する問い合わせは、公式 Discord サーバーにて受け付けます。

わいくらサーバー プライバシーポリシー

改定日：2025年10月13日

第1章 総則

第1条（目的）

本プライバシーポリシー（以下「本ポリシー」）は、当運営が提供するマインクラフトサーバーおよび関連サービスにおけるユーザー情報の取得・利用・管理について定めるものです。ユーザーは本ポリシーに同意したものとみなされます。

第2条（適用範囲）

本ポリシーは、本サービス利用に関連して当運営が取り扱うすべてのユーザー情報に適用されます。

第3条（未成年者の利用）

ユーザーが未成年者である場合、本サービスの利用に当たっては法定代理人の同意を得るものとします。

第2章 収集する情報

第3条（収集情報の種類）

  • 識別情報：Discord ID、Minecraft ID、ユーザー名
  • 利用情報：サーバーへのアクセス履歴、ログイン履歴、チャットや行動ログ
  • 技術情報：IPアドレス、使用端末の種類、クライアントソフト情報
  • その他情報：問い合わせ内容、申請情報など

第3章 利用目的

第4条（利用目的）

  • 本サービスの提供・運営・維持・改善
  • ユーザー認証およびアクセス管理
  • 規約違反・不正行為の防止・対応
  • ユーザーへの連絡やサポート対応
  • サーバー安全確保およびトラブル対応

第4章 情報の第三者提供

第5条（第三者提供の制限）

当運営は、ユーザーの同意なく個人を特定できる情報を第三者に提供することはありません。ただし以下の場合を除きます：

  • 法令に基づく場合
  • 人の生命・身体・財産の保護のために必要な場合
  • 利用規約違反や不正行為の調査に必要な場合

第5章 情報の管理

第6条（安全管理）

当運営はユーザー情報の漏洩、紛失、改ざん、不正アクセスを防止するため、適切な管理体制および技術的措置を講じます。

第7条（保存期間）

ユーザー情報は本サービス運営・法令遵守・紛争対応に必要な期間のみ保持され、不要となった情報は安全に削除されます。

第6章 ユーザーの権利

第8条（権利の行使）

ユーザーは当運営に対し以下の請求を行えます：

  • 自身の情報の閲覧
  • 情報の訂正・追加・削除
  • 利用停止または第三者提供の停止

請求は公式 Discord の問い合わせフォームを通じて行えます。本人確認のためマイクラアカウントとサーバー内システムにリンク済みDiscordアカウントが必要です。

第7章 クッキー・ログの取り扱い

第9条（ログ・統計情報）

サーバー運営のためアクセスログや統計情報を取得する場合があります。個人特定されない形式で集計され、サービス改善やトラブル対応に利用されます。

第8章 プライバシーポリシーの変更

第10条（変更）

必要に応じて本ポリシーを改定できます。改定後は公式Xにて告知され、告知時点から効力を生じます。

第9章 問い合わせ

第11条（問い合わせ窓口）

本ポリシーに関する問い合わせは公式 Discord サーバーにて受け付けます。
